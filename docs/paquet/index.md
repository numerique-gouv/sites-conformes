# Utiliser Sites Conformes en paquet

Cette section s'adresse aux personnes qui veulent **ajouter Sites Conformes à un
projet Django existant**, en l'installant comme paquet Python, plutôt que de
déployer une instance complète à partir du dépôt.

C'est un cas d'usage plus avancé et moins courant : si vous cherchez simplement à
mettre un site en ligne, voir {doc}`../deploiement/index`.
